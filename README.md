
# Comparative Financial Analysis – Automotive Industry (BMW, Mercedes-Benz, Volkswagen)

## 📊 Project Overview

This project presents a comprehensive financial analysis of three leading German automotive companies:
- BMW
- Mercedes-Benz
- Volkswagen

The analysis covers the period **2021–2025** and is built entirely in **Excel**, focusing on comparative performance across profitability, efficiency, leverage, and growth.

The goal of this project is to demonstrate strong financial analysis skills, structured thinking, and advanced Excel modeling capabilities.


---

## 🧱 Model Structure

The Excel model is structured in a clear, layered way to ensure transparency and logical flow:

### 1. Raw Data
- Source financial data for each company
- Includes **Income Statement and Balance Sheet**
- Data harmonized for comparability

### 2. All Raw Data (Harmonized Layer)
- Combines all companies into a **single standardized dataset**
- Ensures consistency across different reporting formats

### 3. Common-Size Analysis
- **Income Statement items → % of Revenue**
- **Balance Sheet items → % of Total Assets**
- Allows structural comparison between companies

### 4. Ratio Analysis
Key financial ratios categorized into:
- **Profitability** (Net Margin, EBIT Margin, Gross Margin, ROA)
- **Efficiency** (Asset Turnover)
- **Leverage** (Equity Multiplier, Debt Ratio)
- **Liquidity** (Current Ratio)

### 5. DuPont Analysis
Breakdown of Return on Equity (ROE) into:
- Net Profit Margin
- Asset Turnover
- Equity Multiplier

### 6. Growth Analysis
Year-over-year growth rates for:
- Revenue
- Net Income
- Total Assets
- Equity

### 7. Final Output & Summary
- Clean, structured output tables
- Designed for readability and quick comparison across companies


---

## 🧮 Key Features

- Fully structured **Excel-based financial model**
- Use of **dynamic formulas (INDEX / MATCH)** for flexible data retrieval
- Clear separation between:
  - Raw data
  - Calculation layers
  - Output presentation
- Harmonization of financial statements across companies for comparability
- Implementation of **DuPont framework** for deeper profitability analysis
